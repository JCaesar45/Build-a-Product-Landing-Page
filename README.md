# Product Landing Page

## Overview

This project is a **Product Landing Page** built to fulfill a set of user stories for a coding challenge. It features a fixed navigation bar, embedded product video, and a subscription form with email validation.

The design is responsive and uses CSS Flexbox for layout and media queries for mobile friendliness. The page does not copy any existing demos but meets all required functionalities.

---

## Features

- Fixed header with a logo image and navigation bar
- Navigation links scroll to corresponding page sections
- Embedded YouTube product video
- Subscription form with:
  - Email input field with HTML5 email validation
  - Placeholder text
  - Submit button that posts email to a mock URL
- Responsive design with media queries
- CSS Flexbox used for layout (nav bar and form)

---

## How to Use

1. Clone or download this repository.

2. Open `index.html` in any modern web browser.

3. Navigate using the fixed top navbar. Click on links to jump to page sections:
   - Features
   - Video
   - Subscribe

4. Watch the embedded product video.

5. Enter your email in the subscription form and click "Subscribe".
   - The email input validates that the text entered is a valid email address.
   - Form submits to `https://www.freecodecamp.com/email-submit` (mock URL).

---

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- Embedded YouTube video (iframe)

## Testing

This project is designed to pass the following user story tests:

- Presence of required HTML elements with specific IDs and classes
- Proper nesting of elements
- Navigation links point to valid sections on the page
- Form validation and submission attributes
- Sticky navbar behavior
- Responsive layout with media queries
- Usage of CSS Flexbox for layout
